Nuevo Proyecto

Apr 17th, 2007

El domingo, al fin, pude iniciar mi nuevo proyecto. Es un proyecto pensado para ayudar a las personas que hacen investigaciones. Por el momento los detalles son secretos ;) pero conforme vaya tomando forma comenzaré a repartir cuentas para los amigos que deseen ser beta testers. En este momento llevo dos días de desvelo pero valen la pena. El primer paso es el más difícil Este proyecto, además de ser un escape al monónoto y poco estimulante trabajo que realizo diariamente, será también un medio de poder ir escribiendo sobre Ruby on Rails. Las características técnicas de este proyecto incluyen:
  • Ruby on Rails Edge
  • PostgreSQL
  • restful_authentication plugin
  • open_id_authentication plugin
  • textile_editor_helpler plugin y acts_as_textiled
  • capistrano
  • mongrel y mongrel_cluster
Los mantendré informados.
