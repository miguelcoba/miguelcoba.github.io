--- layout: post title: Goodbye sun-java5-jdk, welcome openjdk-6-jdk categories: [] tags: - java status: publish type: post published: true meta: _edit_last: '2' --- With Debian Lenny, I finally can install openjdk instead of the official java from sun.com.

As root:

# aptitude install openjdk-6-jdk

This uninstalled (I think, because I don't remember if I first uninstalled the sun-java5-jdk :)) the sun-java5-jdk version and installed the new openjdk-6-jdk.

I updated the alternatives from debian:

# update-java-alternatives -s java-6-openjdk

And finally, to test:

# java -version
java version "1.6.0_0"
OpenJDK  Runtime Environment (build 1.6.0_0-b11)
OpenJDK 64-Bit Server VM (build 1.6.0_0-b11, mixed mode)

Great.
